【ソフトウェア名】ブロック崩し
【対応OS】Windows10 64bit
【製作者】大池陸斗




【本アプリケーションについて】
	本アプリケーションではブロック崩しをプレイすることができます。
	ブロック崩しはパドルという棒を操作しボールを落とさないようにブロックを壊していくゲームです。

【動作に必要なファイルおよびソフトウェア】
	本アプリケーションの動作にはCygwinの各種パッケージ,OpenGLのインストールが必要です。インストール方法は
	http://teacher.nagano-nct.ac.jp/ito/Springs_of_C/cyg64/index.htmlを参考にしてください。

【環境変数の設定】
	Cygwinのインストールフォルダ内のbinを環境変数名"Path"としてパスを通してください。
	Cygwinのインストールフォルダ内のusr\i686-pc-cygwin\sys-root\usr\binを環境変数名"Path"としてパスを通してください。

【インストール方法】	
	本ファイルが入っているディレクトリ内でコマンドプロンプトなどで"make"を実行します。
	なお警告が出る場合がありますが、プログラムの動作に問題はありません。j17404.exeができていれば成功です。
	

【実行方法】
	本ファイルが入っているディレクトリ内でコマンドプロンプトなどで"j17404.exe"を実行するか
	エクスプローラーなどからj17404.exeをダブルクリックで実行してください。
	
【操作方法】
	操作はマウス移動と左ボタンのクリックによって行います。
	タイトル画面、リザルト画面からゲームを始めるときは左ボタンをクリックしてください。
	ゲーム画面においてボールを発射するときは左ボタンをクリックしてください。
	パドルを左右に動かすときはマウスを移動させてください。

【ルール】
	制限時間(300秒)以内にできるだけ早く、多くのブロックを壊してください。
	ボールを下に落としてしまうと残機が1つ減ります。残機は1ゲーム3つです。
	まれにブロックを壊したときにアイテムが落下します。パドルに当たるとアイテムを使う事ができます。
	アイテムはボールやパドルに様々な効果を及ぼします。効果時間は20秒です。
	残機を0にする、制限時間を超過する、ブロックを全破壊するとゲームは終了です。リザルト画面に遷移します。
